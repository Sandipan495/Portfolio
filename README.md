# 📦 Deployment

This website is deployed using **GitHub Pages**.

Steps used for deployment:

1. Push the project to GitHub repository.
2. Go to **Repository Settings → Pages**.
3. Select branch:

```
main / root
```

4. GitHub automatically deploys the website.
